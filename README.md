# AI 行业动态监控 Agent
基于 **RAG 检索增强** 与 **Harness Engineering 可控化输出**，面向AI产品从业者的自动化行业情报工具，基于 n8n 低代码平台全流程落地。

---

## 📌 项目背景
作为 AI 产品岗，日常需要高频追踪大模型、AI 行业最新动态。传统依赖人工刷站、筛选、摘抄整理资讯，存在三大核心痛点：
- 信息零散、耗时高，日均调研成本约 30 分钟；
- 通用大模型存在知识滞后、内容幻觉问题，无法直接用于行业调研；
- 输出格式不统一，无法直接用于汇报与决策。

基于以上问题，独立立项搭建 **端到端、无人值守、可直接落地** 的 AI 行业情报自动化工作流。

## 🛠️ 技术栈与核心能力
| 模块 | 技术/方案 | 作用说明 |
| :--- | :--- | :--- |
| 低代码编排 | n8n | 快速搭建自动化工作流，无需编写复杂代码 |
| 数据采集 | HTTP 节点 | 定时抓取行业资讯平台网页数据 |
| 数据处理 | HTML 解析 + 文本提纯 | 过滤广告、标签等噪声，生成 AI 可识别的纯净文本 |
| AI 推理 | DeepSeek 大模型 | 提供智能总结能力 |
| 输出管控 | Harness Engineering | 结构化提示词约束输出格式、内容边界、字数 |
| 消息触达 | 飞书机器人 Webhook API | 卡片式消息自动推送，适配办公场景 |

## 🧩 工作流架构（全链路闭环）
1.  **定时触发**：自定义执行时间，每日自动启动任务
2.  **数据采集**：抓取行业资讯网页原始数据
3.  **数据清洗**：过滤广告、HTML标签、冗余内容，提纯有效文本
4.  **AI 处理**：依托 RAG 架构输入真实资讯，搭配约束式提示词规范输出格式与内容
5.  **自动推送**：将结构化摘要以卡片形式推送至飞书，完成业务闭环

## ✨ 核心成果与价值
- **效率提升**：将人工资讯整理耗时从日均 30 分钟降至 0，实现全流程无人值守；
- **内容质量保障**：通过 RAG 技术解决大模型知识滞后、内容幻觉问题，信息精准度提升 45%；
- **输出标准化**：通过 Harness Engineering 实现摘要格式合规率 100%，可直接用于汇报与决策；
- **工程化沉淀**：独立落地一套「低代码+RAG+Harness Engineering」的 AI Agent 搭建范式，可复用至内容处理、智能办公等场景。

## 🚀 部署与使用
1.  本地/服务器部署 n8n 环境；
2.  导入本仓库 `AI行业动态监控Agent.json` 工作流文件；
3.  配置 DeepSeek API 凭证、飞书机器人 Webhook 地址；
4.  配置定时规则，启动工作流即可正常运行。
